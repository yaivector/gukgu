# gukgu web v2

เว็บแปลงหน่วยระยะเวอร์ชันพร้อมอัปขึ้น GitHub Pages

## สิ่งที่ปรับแล้ว

- ใช้โลโก้ SVG ใหม่ของ Gukgu
- รองรับภาษาไทยและอังกฤษ
- ตรวจภาษาจากเบราว์เซอร์อัตโนมัติ
- จดจำภาษาที่ผู้ใช้เลือก
- ชื่อหน่วยอยู่บรรทัดเดียว เช่น `เซนติเมตร (cm)`
- รองรับมือถือและคอมพิวเตอร์
- ทำงานออฟไลน์ผ่าน Service Worker
- ติดตั้งเป็นเว็บแอปได้เมื่อเบราว์เซอร์รองรับ
- แปลง mm, cm, m, km, in, ft, yd และ mi

## วิธีอัปเดตเว็บบน GitHub

1. แตกไฟล์ ZIP
2. เปิด Repository เดิมของ Gukgu
3. ลบไฟล์เว็บชุดเก่าหรืออัปโหลดทับ
4. อัปโหลดไฟล์ทั้งหมดที่อยู่ภายในโฟลเดอร์นี้
5. กด Commit changes
6. รอ GitHub Pages อัปเดตประมาณ 1–3 นาที
7. เปิดเว็บและกด Refresh

สำคัญ: ให้อัปโหลด `index.html` ไว้ระดับบนสุดของ Repository ไม่ใช่อัปโหลดทั้งโฟลเดอร์ซ้อนเข้าไป
