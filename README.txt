GUKGU แปลงระยะ V1.0

ไฟล์นี้พร้อมสำหรับนำขึ้นเว็บไซต์แบบ Static Hosting เช่น GitHub Pages,
Cloudflare Pages, Netlify หรือโฮสต์เว็บทั่วไปที่รองรับ HTTPS

ทดลองแบบง่าย:
- ดับเบิลคลิก index.html เพื่อดูและทดสอบเครื่องมือได้ทันที
- การติดตั้งเป็นแอปและระบบออฟไลน์ PWA จะทำงานเมื่อเว็บอยู่บน HTTPS
  หรือเปิดผ่าน localhost เท่านั้น

ไฟล์สำคัญ:
- index.html              หน้าเว็บ
- styles.css              รูปแบบ
- app.js                  ระบบคำนวณ
- manifest.webmanifest    ข้อมูลติดตั้งแอป
- service-worker.js       ระบบออฟไลน์
- assets/gukgu-bird.svg   โลโก้ต้นฉบับ
- icons/                  ไอคอนติดตั้ง

หลักความเสถียร:
- ไม่ใช้ Framework
- ไม่ใช้ไลบรารีภายนอก
- ไม่โหลดฟอนต์ออนไลน์
- ระบบคำนวณไม่พึ่งเซิร์ฟเวอร์
- ความผิดปกติของ Storage, Clipboard หรือ PWA จะไม่ทำให้เครื่องมือหลักหยุดทำงาน
